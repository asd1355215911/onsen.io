Getting Started

Onsen UIを使うと、ハイブリッドアプリやWebアプリのUIを簡単に実装することができます。ハイブリッドアプリを開発する場合は、MonacaやCordova、PhoneGapといったツールを使うことを推奨しています。

Cordovaから使う

コマンドラインから利用する場合は、cordovaツールを使うことが推奨されます。そのためには、node.jsとnpmをあらかじめインストールしておく必要があります。

1. Cordovaのインストール

$ sudo npm install -g cordova

Cordovaツールの使い方については、Cordovaドキュメントを参照してください。

2. テンプレートの選択

ダウンロードページでテンプレートを選択してください。

3. プラットフォームの追加

シミュレータや実機上で動作させるために、プロジェクトにプラットフォームSDKを追加してください。

$ cd /path/to/the/project
$ cordova platform add ios (もしくはandroid)

4. シミュレータでの実行

下記のコマンドでシミュレータ上でアプリを実行できます。

$ cordova emulate

コマンド実行が失敗する場合は、上記の手順が失敗している可能性があります。下記のMonacaを使った方法をぜひお試しください。

Monacaから使う

MonacaでOnsen UIを使うメリットは、クラウド開発が可能になることです。Monacaを使うと、iOSやAndroidアプリ開発を行うために、ローカル環境をセットアップする必要がありません。

1. Monacaの会員登録


http://monaca.mobi     無料サインアップ

2. Onsen UIを使ったテンプレートの選択

IDEを起動して、新規プロジェクトを作って下さい。Onsen UIを使ったプロジェクトテンプレートを選択します。

3. デバッガーのインストールと実行

実機上ですぐにアプリを実行できます。Monacaデバッガーをインストールして、プロジェクトをダウンロードして起動します。

4. AndroidとiOS向けのビルドを行う

iOS、Android、Chrome Apps向けにビルドを行うことが出来ます。アプリをApp StoreやGoogle Playストアに掲載するのも簡単です。

Webサイトで利用する

Onsen UIのBowerモジュールを使うと、簡単にプロジェクトに組み込めます。

$ sudo npm install -g bower
$ bower install onsenui
